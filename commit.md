feat: (Feature) -> Quando você cria uma funcionalidade nova.

fix: (Fix) -> Quando você conserta um bug/erro.

refactor: (Refactor) -> Quando você muda o código para deixá-lo melhor, mas sem criar nada novo e sem consertar bugs 

chore: (Chore/Tarefa) -> Configurações de sistema, instalação de bibliotecas, mudanças no banco que não afetam o usuário final.
